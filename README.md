# Core Python Practice Repository
This repository contains Python practice exercises written in a Google Colab notebook. The purpose of this repository is to build a strong foundation in core Python programming by practicing essential concepts and syntax used in everyday programming.

The notebook includes small code snippets and exercises designed to understand how Python works and how different programming constructs interact with each other.

Topics Covered
1. Python Basics
2. Variables and Data Types
3. Operators
4. Conditional Statements
5. Loops
6. Basic Data Structures
